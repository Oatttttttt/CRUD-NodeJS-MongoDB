## Overview

This application serves as a template for building a RESTful API with basic CRUD functionality using Node.js and MongoDB. It provides endpoints to create, retrieve, update, and delete user information in a MongoDB database. The goal is to demonstrate a simple yet functional implementation of a server-side application with a database backend.

## Features

- **Create User:** Add a new user to the database.
- **Retrieve Users:** Fetch a list of all users or retrieve a specific user by ID.
- **Update User:** Modify user information based on the provided ID.
- **Delete User:** Remove a user from the database.
